# EMO.J 项目全面测试计划

## 一、测试概述

### 1.1 项目简介
EMO.J 是一个基于阿里云百炼API的Web应用，用户上传图片后自动生成6宫格Q版贴纸表情包，支持一键剪切导出和AI抠图功能。

### 1.2 测试范围
- **前端框架**: React 18 + Vite + TypeScript
- **后端服务**: 腾讯云SCF (Node.js)
- **核心功能模块**: 登录、图片上传、提示词编辑、AI生成、结果查看、导出面板
- **部署平台**: 腾讯云 SCF + COS/CloudBase

### 1.3 测试目标
1. 验证所有核心功能模块按需求规格正常工作
2. 确保在不同浏览器和设备上的一致表现
3. 评估系统响应速度和资源占用情况
4. 检查潜在的安全漏洞
5. **确保腾讯云部署的正确性和系统稳定性**
