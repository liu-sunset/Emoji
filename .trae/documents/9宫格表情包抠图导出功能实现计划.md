# 9宫格表情包生成与抠图导出功能实现计划

## 需求概述

将现有的9宫格表情包生成流程变更为：
1. 生成透明背景的9宫格图片（自带分割线）
2. 按分割线自动分割图片
3. 对每个格子进行抠图，提取表情包主体
4. 展示抠图结果
5. 一键导出

## 核心变更

### 1. 修改 Prompt 模板 (`promptConfig.ts`)

**变更内容**：
- 去除背景渐变选项，改为透明背景
- 添加分割线描述
- 保持风格和气泡文字选项

**新的 PROMPT_TEMPLATE**：
```
制作一个 1:1 比例的透明背景 {style} 风格画布，画面包含九个 Q 版贴纸表情包,每行排放3个，共3行，形成标准的3×3九宫格布局。每个格子之间有清晰可见的分割线（白色或浅灰色线条），分割线将画面均分为9个相等的格子。

每个格子内包含一个独立的Q版贴纸，贴纸有着不同的穿搭、姿势和表情。每个贴纸都带有白色边框，并且配有对话气泡，气泡内文字包括：" {bubble1}"、"{bubble2}"、"{bubble3}"、"{bubble4}"、"{bubble5}"、"{bubble6}"、"{bubble7}"、"{bubble8}"、"{bubble9}"。

重要约束：
- 背景必须是完全透明的
- 分割线必须清晰可见，线条宽度约3-5像素
- 每个表情包和气泡文字必须完整落在一个格子内，不能跨越格子边界
- 每个格子之间有明显的间隙用于分割
```

### 2. 修改 PromptEditor 组件 (`PromptEditor.tsx`)

**变更内容**：
- 移除"背景渐变"标签页
- 保留"风格"和"气泡文字"标签页

### 3. 新建图像处理工具 (`src/utils/imageProcessor.ts`)

**功能**：
- `sliceImageByGrid(imageUrl, gridSize)` - 按格子分割图片
- `extractEmojiFromCell(cellImage)` - 从单个格子中抠出表情包主体
- `processAllCells(imageUrl)` - 分割+抠图处理

**抠图算法思路**（因为背景透明）：
1. 加载格子图片
2. 找到所有不透明的像素边界
3. 计算最小包围盒
4. 裁剪出主体区域并添加适当的padding
5. 输出带透明背景的抠图结果

### 4. 修改 ExportPanel 组件 (`ExportPanel.tsx`)

**变更内容**：
- 添加自动抠图功能
- 修改预览展示为显示抠图后的结果
- 添加状态管理：原始图、分割图、抠图结果

**新流程**：
1. 用户点击"生成预览"
2. 自动执行：分割 → 抠图 → 生成9张处理后的图片
3. 展示抠图结果
4. 支持一键下载全部（ZIP）或单独下载某张

## 技术细节

### 抠图算法实现
```typescript
function extractEmojiFromCell(canvas: HTMLCanvasElement): HTMLCanvasElement {
  const ctx = canvas.getContext('2d');
  const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height);
  const data = imageData.data;

  let minX = canvas.width, minY = canvas.height;
  let maxX = 0, maxY = 0;

  // 遍历所有像素，找到不透明像素的边界
  for (let y = 0; y < canvas.height; y++) {
    for (let x = 0; x < canvas.width; x++) {
      const alpha = data[(y * canvas.width + x) * 4 + 3];
      if (alpha > 128) { // 不透明像素
        minX = Math.min(minX, x);
        minY = Math.min(minY, y);
        maxX = Math.max(maxX, x);
        maxY = Math.max(maxY, y);
      }
    }
  }

  // 计算包围盒并添加 padding
  const padding = 10;
  minX = Math.max(0, minX - padding);
  minY = Math.max(0, minY - padding);
  maxX = Math.min(canvas.width, maxX + padding);
  maxY = Math.min(canvas.height, maxY + padding);

  // 创建新画布并绘制抠图结果
  const resultCanvas = document.createElement('canvas');
  resultCanvas.width = maxX - minX;
  resultCanvas.height = maxY - minY;
  const resultCtx = resultCanvas.getContext('2d');
  resultCtx.drawImage(canvas, minX, minY, maxX - minX, maxY - minY, 0, 0, maxX - minX, maxY - minY);

  return resultCanvas;
}
```

## 注意事项

1. 透明背景图片中，分割线应该是可见的（白色或浅灰色）
2. 抠图时需要正确处理边缘，避免切掉表情包边缘
3. 保持导出图片的质量
4. 确保所有状态正确更新
