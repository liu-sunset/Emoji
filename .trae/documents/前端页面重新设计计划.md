# 前端页面重新设计计划

## 任务目标
使用 frontend skill 重新设计 EMO.J 表情包生成器的前端页面，采用：
- **配色方案**：黑色 + 绿色（黑绿配色）
- **设计风格**：苹果公司产品的简约风格
- **布局方式**：弹性布局（Flexible Layout）

## 设计方向

### 配色方案
- **主色调**：深黑色系 (#000000, #1a1a1a, #2d2d2d)
- **强调色**：鲜艳绿色 (#00D26A, #10B981, #34D399)
- **辅助色**：深灰 (#333333, #404040, #666666)
- **文字色**：白色 (#FFFFFF) 和浅灰 (#A0A0A0)
- **边框/分隔**：深灰边框 (#333333)

### 苹果风格特点
- 大圆角设计 (16px-24px border-radius)
- 极简阴影 (subtle shadows)
- 优雅的毛玻璃效果 (backdrop-filter blur)
- 精致的动画过渡 (200-300ms ease)
- SF Pro 字体风格 (使用 -apple-system, BlinkMacSystemFont)
- 大量留白和呼吸空间
- 图标使用线性简约风格

### 弹性布局特点
- 使用 CSS Grid 和 Flexbox
- 响应式断点优化
- 内容自适应容器
- 流动性的间距系统

## 实施步骤

### 1. 更新全局样式 (index.css)
- 定义新的 CSS 变量系统
- 黑绿配色方案
- 苹果风格的基础样式恢复

### 2. 重新设计 Login 登录页面
- 深色背景配合绿色渐变
- 简约居中卡片设计
- 优雅的输入框和按钮
- 震动错误提示动画

### 3. 重新设计 App.css 主布局
- 深色主题导航栏
- 优化的网格布局
- 响应式弹性设计
- 卡片悬浮效果

### 4. 重新设计 ImageUploader 图片上传组件
- 深色拖拽区域
- 绿色虚线边框
- 简约图标设计
- 悬浮预览效果

### 5. 重新设计 PromptEditor 提示词编辑组件
- 深色标签切换
- 绿色锁定徽章
- 优雅的气泡文字编辑网格

### 6. 重新设计 GenerateButton 生成按钮
- 绿色渐变背景
- 简约发光效果
- 优雅的加载动画

### 7. 重新设计 ResultViewer 结果查看组件
- 深色容器背景
- 简约的空状态和加载状态
- 优雅的错误提示

### 8. 重新设计 ExportPanel 导出面板
- 深色卡片设计
- 绿色强调按钮
- 简约的下载网格

## 预期效果
- 整体呈现深色、精致、专业的苹果风格
- 绿色作为点缀强调色，提升视觉层次
- 大量圆角设计增强友好感
- 弹性布局确保各屏幕尺寸下的优秀体验
