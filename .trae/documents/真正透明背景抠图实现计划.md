# 真正透明背景抠图实现计划

## 需求分析

从图1到图2的效果：
- **图1**: 表情包周围有白色边框/背景
- **图2**: 表情包周围完全透明（棋盘格背景表示透明）

当前实现只是裁剪图片边界，但没有去除表情包周围的白色描边/背景。需要使用真正的背景移除技术。

## 技术方案

项目已安装 `@imgly/background-removal` 库（v1.7.0），这是一个基于 AI 的背景移除库，可以实现：
- 自动识别前景主体
- 移除背景并替换为透明
- 保留边缘细节（如头发、衣物边缘）

## 实现步骤

### 第一步：修改 imageProcessor.ts

**新增功能**：
1. 添加 `removeBackground()` 函数 - 使用 @imgly/background-removal 库移除背景
2. 修改 `processAllCells()` 函数 - 对每个切片应用背景移除
3. 修改 `extractBoundsFromCanvas()` 函数 - 在移除背景后重新计算边界

**关键代码**：
```typescript
import { removeBackground } from '@imgly/background-removal';

async function removeImageBackground(blob: Blob): Promise<Blob> {
  const result = await removeBackground(blob);
  return result;
}
```

### 第二步：修改 ExportPanel.tsx

**调整逻辑**：
1. 批量抠图时使用 `removeBackground` 而非简单的边界裁剪
2. 保持现有的 UI 交互不变
3. 优化加载状态提示

### 第三步：添加加载状态优化

背景移除是计算密集型操作，需要：
- 显示"正在移除背景..."的加载提示
- 可能需要显示进度条
- 处理可能的超时情况

### 第四步：测试和验证

- 测试背景移除效果
- 验证透明背景是否正确
- 检查边缘质量

## 验收标准

1. ✅ 表情包周围的白色背景被完全移除
2. ✅ 输出图片为真正的透明背景（PNG格式）
3. ✅ 表情包主体边缘清晰，无锯齿
4. ✅ 批量处理时显示加载进度
5. ✅ 编译检查无错误

## 注意事项

- @imgly/background-removal 库可能需要下载模型文件，首次使用可能较慢
- 建议在本地缓存模型以提高后续处理速度
- 处理大图片时可能需要较长时间，需要添加超时处理
