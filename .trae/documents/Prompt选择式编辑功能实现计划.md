# Prompt 选择式编辑功能实现计划

## 需求概述

将 PromptEditor 从自由文本输入改为通过点击选择的方式生成 prompt，用户可选自定义风格、气泡文字和背景渐变。

## Prompt 模板分析

默认 prompt 模板包含 3 个可替换变量：

* `{卡通风}` → 风格选择

* `{早上好}|{咦}|{抱抱}|{谢谢}|{晚安}|{你超棒的}|{想你}|{亲亲😙}|{做得好}` → 9个气泡文字

* `{柔和的白色到浅蓝渐变}` → 背景渐变颜色

## 实现步骤

### 1. 创建 Prompt 配置数据文件

**文件**: `emo-j/src/utils/promptConfig.ts`

定义：

* **风格选项列表** (默认: 卡通风)

  * 卡通风、迪士尼风格、像素风格、水彩画风格、扁平插画风格、复古漫画风格、赛博朋克风格

* **气泡文字预设** (默认9个)

  * 预设组1: 早上好、咦、抱抱、谢谢、晚安、你超棒的、想你、亲亲😙、做得好

  * 预设组2: 你好呀、哈哈、加油、爱你、么么哒、辛苦啦、收到、好棒、晚安

  * 预设组3: 笑死、离谱、救命、好耶、太卷、摸鱼、躺平、摆烂、冲鸭

  * 用户可自定义输入

* **背景渐变预设** (默认: 柔和的白色到浅蓝渐变)

  * 柔和的白色到浅蓝渐变

  * 温暖橙色到粉色渐变

  * 清新的绿色到青色渐变

  * 浪漫紫色到粉色渐变

  * 经典蓝色到紫色渐变

  * 日落橙色到黄色渐变

  * 薄荷绿到蓝色渐变

  * 玫瑰红到橙色渐变

### 2. 修改 PromptEditor 组件

**文件**: `emo-j/src/components/PromptEditor/PromptEditor.tsx`

新增：

* Tab 切换：选项选择 | 自定义编辑

* 风格选择区块：网格展示，点击选中

* 气泡文字区块：

  * 预设组选择器（点击切换预设组）

  * 每个气泡支持点击编辑

* 背景渐变区块：网格展示，点击选中

状态管理：

* `selectedStyle: string` - 选中的风格

* `selectedBubbleTexts: string[]` - 选中的气泡文字数组

* `selectedGradient: string` - 选中的背景渐变

* `activeSection: 'style' | 'bubbles' | 'gradient'` - 当前编辑区块

### 3. 修改 PromptEditor CSS

**文件**: `emo-j/src/components/PromptEditor/PromptEditor.css`

新增样式：

* 选项卡片网格布局

* 选中状态的视觉反馈（边框高亮、绿色背景等）

* 预设组切换器样式

* 响应式布局适配

### 4. 更新 App.tsx 中的状态管理

**文件**: `emo-j/src/App.tsx`

修改：

* `prompt` 状态初始化为完整模板字符串（带默认值）

* 不再需要单独的 `bubbleTexts` 状态

* `handleGenerate` 中直接使用组装好的 prompt

### 5. 构建最终 Prompt 的逻辑

组装函数 `buildPrompt(style, bubbleTexts, gradient)`：

```
"制作一个 10:16 比例的 3D {style}画布，画面包含九个 Q 版贴纸,每行排放3个，贴纸有着不同的穿搭、姿势和表情。请参考附带的图片进行创作。每个贴纸都带有白色边框，并且配有对话气泡，气泡内文字包括："{bubble1}"、"{bubble2}"、"{bubble3}"、"{bubble4}"、"{bubble5}"、"{bubble6}"、"{bubble7}"、"{bubble8}"、"{bubble9}""。背景采用{gradient}效果，营造出活泼、积极的氛围。注意生成的9个表情之间要留一些空隙,后续我会按照3*3分割这个图片,保证分割之后的单个表情包的完整性"
```
