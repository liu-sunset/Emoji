# 图片尺寸规格与裁切策略优化计划

## 任务目标
1. 将生成图片的尺寸规格从比例描述改为固定 2000px * 2000px
2. 约束单个表情包和气泡文字的宽高不超过 500px
3. 相邻表情包间距不低于 80px
4. 优化裁切策略保证表情包完整性

## 实施步骤

### 1. 修改 promptConfig.ts - 更新 PROMPT_TEMPLATE
- 将 "1:1 比例" 改为 "2000px * 2000px 固定像素尺寸"
- 添加单个表情包宽高不超过 500px 的约束
- 添加相邻表情包间距不低于 80px 的约束
- 调整 prompt 描述以确保 AI 能正确生成符合规格的图片

### 2. 修改 imageProcessor.ts - 优化裁切策略
- 改进 extractBoundsFromImageData 函数，增加边界扩展逻辑
- 确保裁切时保留足够的 padding 区域
- 增加对表情包主体完整性的检测，确保不截断表情包
- 考虑在边界检测时加入膨胀(dilation)操作，防止裁切过于接近表情包边缘

## 改动详情

### promptConfig.ts 改动
```
原: 制作一个 1:1 比例的透明背景 {style} 风格画布
改: 制作一个 2000px * 2000px 固定尺寸的透明背景 {style} 风格画布

添加约束:
- 每个格子内的 Q 版贴纸和气泡文字的宽高都不能超过 500px
- 上下左右相邻的表情包之间保持至少 80px 的间距
```

### imageProcessor.ts 改动
- 优化 cropCanvasToBounds 函数
- 扩大 padding 值从 15 提升到 40（确保 80px 间距要求）
- 增加边界完整性检测，确保裁切不会截断表情包主体
