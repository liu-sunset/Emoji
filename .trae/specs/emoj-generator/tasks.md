## 技术栈详情

### 前端依赖
- react ^18.2.0
- react-dom ^18.2.0
- typescript ^5.0.0
- vite ^5.0.0
- jszip ^3.10.0
- file-saver ^2.0.5

### 后端依赖（云函数）
- axios ^1.6.0（API调用）

### 环境变量

**前端 (.env)**
```
VITE_API_BASE_URL=https://your-scf-url.tencentyun.com
```

**云函数环境变量**
```
DASHSCOPE_API_KEY=您的API密钥
ACCESS_PASSWORD=访问密码
DAILY_LIMIT=10
```
